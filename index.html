Presencia de los SVG en la web

Los SVG (Scalable Vector Graphics) están presentes en muchísimos recursos de la web porque son un formato ligero, escalable y perfecto para gráficos que no pierden calidad.

Dónde encontrarlos

Como recursos en la interfaz gráfica para el diseño y desarrollo web

  • Iconos : Font Awesome, Heroicons, Feather Icons, Material Icons suelen ofrecer versiones en SVG.
  • Ilustraciones: Páginas como undraw.co ofrecen ilustraciones en SVG para usar libremente.
  • Animaciones : hay librerías como https://www.svgator.com/ que trabajan con svg
  • Fondos y patrones
  • Librerías con chart.js usan svg para renderizar los gráficos
  • La mayoría de logotipos en la actualidad se muestran en svg
  • Existen una gran cantidad de bancos gratuitos : flaticon, svg repo, pixebay, repositorios de github....
